# MullV Earthworks

MullV Earthworks is the official modpack of the **mullv.studio** network, built for **Minecraft 1.21.1** on **NeoForge**, featuring a custom **1:1 terrain generation** world and a polished, performance‑focused vanilla+ experience.

This pack combines high‑quality client utilities, strong performance mods, immersive visuals, and a large Create ecosystem, all tuned for long‑term multiplayer survival on the MullV network.

![banner](./banner.png)

---

## Key Features

- **1:1 custom terrain world**
  - Explore a custom, Earth‑like 1:1 terrain generation used on the official MullV servers for large‑scale builds, exploration, and infrastructure projects.

- **Create‑centric engineering**
  - Core Create plus a full suite of addons:
    - Create Big Cannons, Crafts & Additions, Diesel Generators, Dynamic Light, Nuclear, Ore Excavation, Railways Navigator, Train Physics, Train Utilities, Bells & Whistles, Interiors, New Age, Steam 'n' Rails, Structures Arise, Threaded Trains, Trading Floor, Waystones Recipes and more.
  - Build automated factories, logistics networks, train systems, and large‑scale industry that integrate smoothly into the MullV world.

- **Exploration & world flavor**
  - Antique Atlas, Ecologics, Repurposed Structures, Traveler's Titles, Surveystones, Surveyor Map Framework, Realistic Bees, Immersive Aircraft, Handcrafted, Macaw’s Bridges/Furniture/Roofs, Immersive Paintings, Exposure & addons, and more for richer exploration and building.

- **Multiplayer‑ready utilities**
  - Simple Voice Chat, Gravestone Mod, Hardcore Revival, Waystones, Tom’s Simple Storage, Carry On, Double Doors, FallingTree, Easy Anvils, Create Ore Excavation and other QOL enhancements tailored to co‑op survival.

- **Performance & stability**
  - Sodium, Lithium, Sodium Extra/Extras, Sodium Dynamic Lights, ImmediatelyFast, ModernFix, FerriteCore, CreateBetterFps, Saturn, EntityCulling, Dynamic FPS, ThreatenGL, Distant Horizons and more for a smooth experience on low and high‑end machines.
  - Crash Assistant, Modpack Update Checker, Euphoria Patcher, Global Packs, Load Support and several libraries ensure stability and easier maintenance.

- **Immersive visuals & UI**
  - Iris (shaders), AmbientSounds, EnhancedVisuals, Immersive Thunder, Sound Physics Remastered, Blur+, ImmersiveUI, SpiffyHUD, BetterF3, Entity Texture/Model Features, Tiny Item Animations, WaveyCapes, oωo and more provide a modern, cohesive visual overhaul.

- **Client quality‑of‑life**
  - EMI, Jade + Jade Addons, Better Advancements, Advancement Plaques, Advancement Screenshot, AppleSkin, Legendary Tooltips, Item Borders, Highlighter, Colorwheel, Controlify, Mouse Tweaks, Just Zoom, Chat Heads, ChatAnimation, Searchables, Pick Up Notifier, Visual Workbench, No Chat Reports, etc., for better information, controls, and workflows.

## Installation

1. Install **Minecraft 1.21.1**.
2. Install a **NeoForge‑compatible** mod loader setup (or a launcher that supports NeoForge and Modrinth modpacks).
3. Download the MullV Earthworks pack from the Modrinth page or import the `.mrpack` into your launcher.
4. Allocate at least **6–8 GB of RAM** for a smoother experience, especially with shaders and Distant Horizons.
5. Join the official MullV server using the IP provided by mullv.studio.

---

## Recommended Settings

- Enable **Iris** for shaders (optional but recommended).
- Configure **Distant Horizons** and **Sodium** to your hardware; start with default presets, then tweak.
- Use **EMI**, **Jade**, and **Waystones** keybinds; adjust via Controls → Key Binds.
- For controllers, configure **Controlify** in its dedicated settings menu.
